# SSA Dashboard

Aplikasi desktop native untuk mengotomatisasi pengolahan data laporan dashboard perbankan (Bank BRI — AH Gunsar Jakarta Region).

## Persyaratan
- Python 3.11+
- Install dependensi: `pip install -r requirements.txt`

## Cara Menjalankan
`python main.py`

## Build .EXE (Windows)
`pyinstaller build.spec`
